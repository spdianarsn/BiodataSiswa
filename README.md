# 📋 Biodata Siswa

Aplikasi Biodata Siswa sederhana yang dibuat menggunakan HTML, CSS, dan JavaScript.

## ✨ Fitur

- ✅ Tambah data siswa
- ✅ Edit data siswa
- ✅ Hapus data siswa
- ✅ Validasi input
- ✅ Modal detail
- ✅ Tampilan modern (Glassmorphism)

## 🛠️ Teknologi

- HTML5
- CSS3
- JavaScript

## 📂 Struktur Project

```
BiodataSiswa
│
├── css
│   └── style.css
│
├── js
│   └── script.js
│
└── index.html
```

## 🚀 Cara Menjalankan

1. Download repository.
2. Buka folder menggunakan Visual Studio Code.
3. Jalankan menggunakan Live Server.
